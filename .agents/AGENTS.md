# User Work Preference Rule

<RULE[user_work_preference]>
The user will NOT do any technical work, including running development servers (like `pnpm run dev`), testing endpoints, executing builds, or debugging technical issues. 
As the agent, you must take full responsibility for technical execution. Do not ask the user to run commands or start servers. You must do it yourself using the `run_command` tool and verify the results independently.
</RULE[user_work_preference]>
