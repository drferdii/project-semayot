AI summary internal agent (re-use pola chat-widget) + SEO panel rekomendasi yang di-generate untuk owner apply manual.

5.1 · Mekanisme AI Summary
Trigger · data source · prompt · cache · output
① Trigger
On-demand: tombol "Generate ringkasan" di /admin/ai. Tidak real-time per transaksi (terlalu berisik).
② Data source
Transaksi + expenses + top menu items dari Supabase. Range: hari ini / 7 hari / 30 hari.
③ Prompt + AI
System prompt admin-specific (owner-facing). Model: mistral-7b via OpenRouter. Streaming response.
④ Cache
Hasil di-simpan di tabel ai_summaries per periode. Tombol "Refresh" untuk regenerate.

5.3 · Mekanisme SEO Panel

Sumber rekomendasi

Statis (curated oleh OG) — 10 rekomendasi berdasarkan best practice SEO lokal resto: meta tags, JSON-LD Restaurant, sitemap, robots, OG image, heading audit, performance budget, internal linking, Google Business Profile, mobile-friendly.

AI-generated (masa depan) — OG agent bisa scan public site via fetch + analisis HTML, tapi ini Phase B+ (overkill untuk MVP).

Alur kerja

1. Admin tampilkan list 10 rekomendasi statis.
2. Tiap item: judul, kenapa penting, kode/nilai siap salin, lokasi apply, status.
3. Owner klik "Salin" → paste manual ke public site.
4. Owner tandai "Sudah diterapkan" (status only, no auto-verify).
5. Rekomendasi di luar list: form "Tambah rekomendasi sendiri" (owner/konsultan SEO eksternal).

5.4 · Halaman /admin/seo (panel rekomendasi)
10 rekomendasi SEO · 3 diterapkan · 7 menunggu
✓ TERAPAN
1. Meta title & description
Title dan description saat ini sudah di-set di app/layout.tsx. Panjang optimal 50-60 char untuk title, 150-160 untuk description.
Lihat kode saat ini 
title: "Rumah Makan Semayot | Cita Rasa Khas Dayak..."
description: "Nikmati kehangatan hidangan..."
⏳ MENUNGGU
2. JSON-LD Restaurant schema
Tambah blok <script type="application/ld+json"> di app/layout.tsx untuk Google rich results (buka jam, rating, alamat, menu).
⏳ MENUNGGU
3. Sitemap.xml
Generate public/sitemap.xml lalu submit ke Google Search Console. Untuk single-page site, sitemap berisi 1 URL + lastmod.
+ Lihat 7 rekomendasi lain (robots.txt, OG image, heading audit, perf budget, dll)
4. robots.txt — izinkan semua crawler, referensikan sitemap.
5. OG image (1200×630) — buat 1 gambar untuk share preview. Rekomendasi: foto Daging Asap Suwir + logo.
6. Heading hierarchy audit — pastikan 1× H1, berjenjang ke H2/H3.
7. Performance budget — LCP < 2.5s, CLS < 0.1, INP < 200ms. Sudah dicek via Lighthouse.
8. Google Business Profile — klaim & verifikasi listing, upload foto mingguan.
9. Local keywords — "Rumah Makan Dayak Bengkayang", "Makan Siang Bumi Amas".
10. Internal linking — link WhatsApp CTA ke nomor, link Maps ke embed.
