4.4 · 9 Halaman /admin/*

/admin/login

Email + password form. Brand warm cream. Lupa password → Supabase recovery email. Redirect ke overview setelah berhasil.

/admin/overview

Dashboard owner. 4 stat cards (pendapatan, transaksi, menu terlaris, pengeluaran) + tabel 5 transaksi terakhir. Staff lihat versi ringkas (hanya transaksi sendiri).

/admin/menu

Owner only. List menu + tombol Tambah/Edit/Hapus. Form: nama, harga, kategori, foto (upload), badge, status aktif. Soft delete.

/admin/pos

Staff + owner. POS form (mockup di atas). Cari menu, tambah ke keranjang, input tunai, hitung kembalian otomatis, catat.

/admin/transactions

Owner (semua) + staff (sendiri). List transaksi + filter (tanggal, staff, range). Klik row → detail modal dengan line items.

/admin/reports

Owner only. Tab: Harian | Bulanan. Grafik line pendapatan, tabel breakdown menu, pie expense by category. Tombol Export CSV/PDF.

/admin/ai

Owner only. Panel AI summary harian/mingguan/bulanan (cached di DB). Chat dengan AI agent untuk tanya analitik (mirip chat-widget tapi internal).

/admin/seo

Owner only. List rekomendasi SEO yang OG-generated (meta tags, JSON-LD, OG, perf, dsb). Setiap item ada tombol "Salin" untuk di-paste manual ke public site.

/admin/settings

Owner only. Tab: Bisnis (jam buka, alamat, telepon, WhatsApp) | Staff (list + invite + nonaktifkan). Catatan: ini TIDAK replace data di public site, hanya referensi untuk owner.

4.5 · Komponen Reusable (components/admin/)

AdminShell

Layout wrapper (sidebar + topbar + main). Terima role untuk hide nav items.

StatCard

Card statistik dengan label, value, delta. 4 varian warna (default, success, danger, accent).

DataTable

Tabel generik dengan sort, filter, pagination. Slot untuk action buttons per row.

FormField

Input wrapper: label, input/textarea/select, error message, hint. Reuse dengan react-hook-form + zod.

Modal

Dialog modal dengan backdrop. Trigger via context. Support form di dalam (untuk add/edit).

Toast

Notifikasi sukses/error/info. Auto-dismiss 3 detik. Stack di pojok kanan atas.

ConfirmDialog

Modal konfirmasi untuk aksi destruktif (hapus menu, nonaktifkan staff). Tombol cancel + confirm.

EmptyState

Empty state dengan mascot variant loading, pesan friendly, CTA. Reuse untuk list kosong.

ErrorBoundary

React error boundary dengan fallback UI yang menampilkan pesan error + tombol reload.

DateRangePicker

Picker range tanggal untuk filter laporan. Default: bulan ini. Quick select: hari ini, 7 hari, 30 hari.

MoneyDisplay

Format angka ke Rupiah. Terima cents (int) → render "Rp 45.000". Konsisten di seluruh app.

FileUpload

Upload foto menu. Drag-drop, preview, validasi type/size. Upload ke Supabase Storage, return public URL.
